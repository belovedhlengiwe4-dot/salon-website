# Luxury Hair & Beauty Salon Website

This is a responsive 3-page salon website built using HTML and CSS.

## Pages Included
- Home page
- Services page
- Contact page

## Features
- Navigation bar
- Responsive layout (mobile-friendly)
- Flexbox layouts
- Styled buttons and cards
- Contact form design
- Image sections
- Service pricing layout

## Purpose
This project was built as part of my journey learning web development.  
It focuses on improving my HTML and CSS skills, especially layout design using Flexbox.

## What I Learned
- Structuring multi-page websites
- Using Flexbox for layouts
- Creating responsive designs with media queries
- Styling navigation bars, cards, and forms
- Organizing files and folders correctly for deployment
